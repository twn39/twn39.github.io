安装 Elixir

Jun 20th, 2014 | Comments

Elixir是一种运行于ErlangVM上具有元编程特性的动态语言。它拥有灵活的语法和强大的宏系统， 还具备模式匹配、通过协议（类似Clojure的方式）支持多态等功能：继承了Erlang的优点，尤其适合开发并行、分布式，以及高容错应用。
