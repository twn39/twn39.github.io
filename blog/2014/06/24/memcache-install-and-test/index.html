在 Ubuntu 中安装 Memcached

Jun 24th, 2014 | Comments

Memcached是一个流行的高性能分布式的内存对象缓存系统， 通常被用于降低数据库加载压力以提高动态web应用的响应速度。 目前许多的大型企业都有使用 memcache 来作为缓存数据库。

以下讲解如何在 ubuntu 上安装 memcached 。

安装

一. 安装 memcached 服务

默认的配置为 localhost：11211

二. 安装 php 插件

memcache 的 php 插件有两个，一个是php5-memcache, 另一个为php5-memcached。好像前一个要出现得早一些， 但是后者能够提供更多的功能，因此这里安装后者。

重启 apache2

测试

新建两个页面，一个添加缓存数据，另一个获取缓存数据。

可以发现两个页面都可以正常执行，说明 memcache 安装成功。
