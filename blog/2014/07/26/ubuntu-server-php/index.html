Ubuntu Server:PHP,Nginx,MySQL,NoSQL,Beanstalkd配置与安装

Jul 26th, 2014 | Comments

搭建 LAMP 环境是一件非常繁琐的事情，尤其是随着系统的不同，相应的配置也会有所差异。 以下是 LAMP 环境和一些其他服务的安装与配置，基于 Ubuntu 14.04 LTS。可用于开发环境， 因为本身不是运维人员，也不知道能否用于生成环境。

PHP 与 Nginx

安装： 

sudo apt-get install nginx php5 php5-fpm php5-cli

Nginx 在 Ubuntu 中的网页根目录为 /usr/share/nginx/html, 配置文件为 /etc/nginx/。

重启 Nginx：

sudo service nginx restart

安装扩展：

搜索扩展：

apt-cache search php5-

安装扩展：

sudo apt-get install php5-curl php5-mcrypt php5-apcu php5-imagick php5-oauth

像 mcrypt 需要手动启用，启用扩展：

sudo php5enmod mcrypt

重启Nginx：

sudo service nginx restart

MySQL

命令安装： 

sudo apt-get install mysql-server mysql-client

在安装过程中会提示输入密码，输入密码就可以继续往下安装。

查看进程：

ps -aux | grep mysql

默认端口为：localhost:3306

安装PHP扩展：

sudo apt-get install php5-mysql

启动/停止/重启

sudo service mysql start/stop/restart

进入shell环境：

Memcache

命令安装：

sudo apt-get install memcached

php扩展：

sudo apt-get install php5-memcached

查看进程：

ps -aux | grep memcached

默认端口：localhost：11211

Redis

命令安装:

sudo apt-get install redis-server redis-tools

PHP扩展：

sudo apt-get install php5-redis

默认端口：localhost:6379

进入shell环境：

MongoDB

安装：

sudo apt-get install mongodb mongodb-clients

php扩展：

sudo apt-get install php5-mongo

启动/停止/重启：

sudo service mongo start/stop/restart

默认端口：localhost:27017

使用shell环境：

Beanstalkd

sudo apt-get install beanstalkd

默认端口：localhost：11300
