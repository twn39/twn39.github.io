使用pandas处理数据

Apr 17th, 2014 | Comments

Python 现在越来越多地被用于处理数据与科学计算，其中比较有名的有Numpy和Scipy。但是随着技术的发展，越来越多的优秀的扩展被开发出来，像IPython，Pandas，Seaborn等等。

使用Pandas包可以非常轻松地对数据进行导入与清洗。以下使用Ipython QT处理数据：

数据导入函数: read_csv()

如图所示：

获取一列数据：

col = data["sepal length"]
print col

或者：

col = data[[2]]
print col

获取多列数据：

cols = data[["sepal length", "sepal width"]]
print cols

或者：

cols = data[[2,4]]
print cols

获取一行

row = data[3:4]
print row

获取多行数据：

rows = data[3:9]
print rows

组合：

取出单个数据：

a = data[2:3]
b = a[[3]]
print b

取出矩形数据

a = data[[1,3,4]]
b = a[4:8]
print b

时间序列处理 导入数据：
