javascript模板引擎:Handlebars.js

Jul 13th, 2014 | Comments

用过 PHP 框架的人应该基本都知道模板引擎，MVC 模式中的视图(view)。视图将逻辑与表现分离开来， 这样做的好处就是使代码更易于协作与维护，而实现这种功能就需要模板引擎了。

然而 PHP 框架中的模板引擎基本都是使用 PHP 实现的，即是服务端模板引擎，但是这里将介绍的是客户端模板引擎 ，即 JavaScript 模板引擎，它的优势就是减轻了服务端的压力，不过它也有一些缺点，就我所知，它对搜索引擎不太友好。

目前有许多 javascript 模板引擎，不过我比较喜欢的是handlebars.js
