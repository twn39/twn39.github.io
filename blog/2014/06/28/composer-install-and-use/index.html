Jun 28th, 2014 | Comments

composer 是目前非常流行的管理 php 依赖包工具，目测现在啥都得有个包管理工具 才行，像 web 开发都有 bower 了。不过使用 composer 确实方便了不少，如果像是个人 或者小项目直接用 composer 装几个模块就可以开发了，而且近年来风头正劲的 php 框架 Laravel 也使用 composer 来处理依赖了。所以学习 composer 是非常有用的。
